### 🎯 Миссия проекта
**Centum** — это прикладной инструмент для управления личным благосостоянием. Мы переложили мудрость классической финансовой философии на язык строгих математических алгоритмов, чтобы создать надежный фундамент для вашего капитала.

---

## 🛠 Методология и Философия

| Принцип | Описание |
| :--- | :--- |
| **⚙️ Технократизм** | Решения на основе математики. Эффективность и точность — наш приоритет. |
| **👤 Ответственность** | Мы верим в экономически стабильного и независимого гражданина. |
| **🛡 Суверенитет** | Полная приватность. Данные хранятся только на вашем устройстве. |

### 🔒 Безопасность и Приватность
*   **Offline-first:** Никаких внешних серверов. Ваша финансовая тайна защищена физически.
*   **Прозрачность:** Исходный код открыт для аудита — мы гарантируем отсутствие скрытого контроля.

---

## 📦 Практическое подтверждение: «Skin in the Game»

Разработка Centum — это не теория. Основатель проекта проверил эффективность алгоритмов лично:

> "Чтобы доказать работоспособность системы, я прошел путь практического освоения финансовых инструментов, работая в реальном секторе (логистика и доставка). Результатом стало формирование портфеля **государственных облигаций (ОФЗ)**, который уже приносит пассивный доход."

**Вывод:** Если дисциплина и верные алгоритмы работают здесь, они сработают для каждого.

## 🚀 Основные возможности

*   **💰 Портфель сбережений:** Автоматический расчет доли дохода «для себя».
*   **⚖️ Баланс бюджета:** Четкое разделение на траты и развитие.
*   **📈 Психологический комфорт:** Интерфейс, мотивирующий на созидание и рост.
*   **🤖 Smart-помощник (In Dev):** Локальный ИИ для анализа стратегий без передачи данных в сеть.
